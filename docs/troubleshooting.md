# Troubleshooting

## Common Issues

### Canary Timeout
- Increase timeout setting
- Check target endpoint performance
- Optimize script wait conditions

### False Positives
- Add retry logic
- Increase wait timeouts
- Check for flaky selectors

### High Latency Alerts
- Verify baseline metrics
- Check from multiple locations
- Review network path

### Screenshot Failures
- Check viewport settings
- Verify element visibility
- Review page load timing
